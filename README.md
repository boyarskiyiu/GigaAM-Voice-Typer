# GigaAM Complete — Голосовой ввод

**Версия 1.6.0** | (c) Боярский Игорь Юрьевич, 2026

Программа для распознавания русской речи и автоматической вставки текста в активное окно.

## 🚀 Возможности

- Распознавание речи моделью GigaAM-v3 (Сбер).
- Исправление опечаток и повторов через Яндекс.Спеллер.
- Автоматическая установка всех зависимостей и модели при первом запуске.
- Автоматический выбор микрофона и калибровка уровня шума.
- Проверка обновлений через GitHub.
- Горячие клавиши: `F2` — пауза, `F3` — исправить фразу, `F4` — свернуть.

## 📥 Установка и запуск

1. Скачайте файл `GigaAM_Complete.pyw` из раздела [Releases](https://github.com/boyarskiyiu/GigaAM-Voice-Typer/releases/latest).
2. Убедитесь, что установлен Python 3.8–3.11.
3. Запустите файл двойным щелчком. При первом запуске программа скачает модель (около 500 МБ) и все необходимые библиотеки.
