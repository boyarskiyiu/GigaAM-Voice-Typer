## ✨ Что нового в 3.0.0 (революционное обновление)

- **Гибридное распознавание**: VOSK обеспечивает **непрерывный потоковый ввод** (текст появляется сразу во время речи), а GigaAM-v3 подключается в конце фразы для **максимальной точности**.
- **Полная автономность**: Модель VOSK (~40 МБ) автоматически скачивается при первом запуске. Интернет нужен только один раз.
- **Современный интерфейс**: Программа полностью переведена на `ttkbootstrap` с темой **Flatly** – выглядит стильно и аккуратно.
- **Улучшенная стабильность**: Устранены все известные причины зависаний, добавлен пинг аудиопотока и автоматическое восстановление горячих клавиш.
- **Умный офлайн-спеллер**: `pyspellchecker` исправляет опечатки без доступа к сети.

## 📥 Установка

### 🚀 Быстрый запуск
1. Скачайте **`GigaAM_Complete.pyw`** из [раздела Releases](https://github.com/boyarskiyiu/GigaAM-Voice-Typer/releases/latest).
2. Поместите его в любую папку (например, на рабочий стол).
3. Запустите двойным щелчком.
   - При первом запуске программа автоматически скачает и установит все необходимые компоненты, включая модель VOSK (~40 МБ).
4. Говорите – текст будет появляться в реальном времени, а после паузы корректироваться моделью GigaAM.
