# 🎤 GigaAM Complete — Голосовой ввод с ИИ

![Версия](https://img.shields.io/badge/version-2.0.17-blue)
![Лицензия](https://img.shields.io/badge/license-MIT-green)
![Платформа](https://img.shields.io/badge/platform-Windows%2010%2F11-lightgrey)

**GigaAM Complete** — программа для распознавания русской речи в реальном времени с автоматической вставкой текста в любое активное окно. Использует модель **GigaAM-v3** от Сбера (точность ~3.3% WER) и **Яндекс.Спеллер** для исправления опечаток.

## ✨ Возможности

- 🎯 Высочайшая точность распознавания русского языка
- 🔄 Автоматическая вставка текста в активное приложение (Word, браузер, блокнот и др.)
- 📚 Сохранение исправлений для повторяющихся ошибок
- 🪄 Яндекс.Спеллер исправляет опечатки и повторы
- ⚡ Горячие клавиши: **F2** — пауза/продолжить, **F3** — исправить последнюю фразу, **F4** — свернуть
- 🌐 Окно всегда поверх других (можно отключить)
- 📦 Автоустановка всех зависимостей и модели при первом запуске

## 📥 Установка и запуск

### Способ 1: Автоматическая установка (рекомендуется)

1. Скачайте файл `Setup_GigaAM.bat` из [раздела Releases](https://github.com/boyarskiyiu/GigaAM-Voice-Typer/releases).
2. Запустите его от имени администратора.
3. Скрипт сам удалит старые версии, установит Python 3.11.9, все библиотеки и создаст ярлык на рабочем столе.

### Способ 2: Ручная установка (для разработчиков)

1. Убедитесь, что установлен Python 3.9–3.11 (рекомендуется 3.11.9).
2. Клонируйте репозиторий или скачайте `GigaAM_Complete.pyw`.
3. Установите зависимости: `pip install onnxruntime onnx "onnx-asr[cpu,hub]" sounddevice numpy keyboard scipy pyperclip librosa pyautogui pillow pyaspeller requests`
4. Запустите `pythonw GigaAM_Complete.pyw`.

## 🎮 Использование

1. Убедитесь, что микрофон подключён и работает.
2. Запустите программу.
3. Нажмите **F2** или кнопку «⏯ Возобновить», чтобы начать прослушивание.
4. Говорите в микрофон — текст будет появляться в активном окне.
5. Чтобы исправить последнюю распознанную фразу, нажмите **F3** и введите правильный вариант. Программа запомнит исправление.

## ⚙️ Настройка

- Файл `corrections.json` хранит пользовательские исправления.
- Чтобы отключить режим «поверх всех окон», закомментируйте строку `self.root.attributes('-topmost', True)` в коде.

## 🔧 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Программа не запускается, ошибка `ModuleNotFoundError` | Запустите `Setup_GigaAM.bat` от имени администратора |
| Микрофон не определяется | Проверьте настройки конфиденциальности Windows для микрофона |
| Текст не вставляется | Убедитесь, что целевое окно активно; попробуйте вставить вручную (`Ctrl+V`) |
| Яндекс.Спеллер не работает | Проверьте интернет-соединение; если ошибка повторяется, программа продолжит работу без спеллера |
