## 📖 О программе

GigaAM Complete — это программа для **голосового ввода текста** на Windows. Она позволяет надиктовывать текст в любое активное окно (Word, Блокнот, браузер и т.д.) без необходимости нажимать кнопки мыши или клавиатуры. Программа работает **полностью офлайн** после первой загрузки модели, обеспечивая высокую точность распознавания русской речи.

### ✨ Ключевые возможности

- **Высокоточное распознавание речи** на базе нейросетевой модели **GigaAM v3** (RNNT).
- **Активация по голосу (VAD):** программа начинает распознавание только тогда, когда вы говорите, и автоматически завершает фразу после паузы.
- **Умный фильтр междометий:** автоматически удаляет из текста «э-э», «м-м» и другие паразитные звуки.
- **Проверка орфографии:** через **Яндекс.Спеллер** (требуется интернет).
- **Горячие клавиши:** F2 — пауза/возобновление, F3 — исправление последней фразы, F4 — свернуть окно.
- **Компактный интерфейс:** всегда поверх других окон, с логом работы и полем для просмотра последней фразы.
- **Автоустановка зависимостей:** при первом запуске программа сама установит необходимые библиотеки.

## ⚙️ Системные требования

- **ОС:** Windows 10/11 (x64)
- **Python:** 3.11 (рекомендуется)
- **ОЗУ:** ≥ 4 ГБ
- **Свободное место на диске:** ~2 ГБ (для модели GigaAM)
- **Микрофон:** любой подключенный и настроенный

## 🚀 Быстрый старт

1. **Установите Python 3.11** с официального сайта (отметьте «Add Python to PATH»).
2. **Скачайте файл `GigaAM_Complete.pyw`** из [Releases](https://github.com/boyarskiyiu/GigaAM-Voice-Typer/releases).
3. **Запустите файл** двойным кликом.
4. При первом запуске программа автоматически:
   - установит необходимые библиотеки (`onnx-asr`, `sounddevice`, `numpy` и др.);
   - загрузит модель GigaAM (~500 МБ).
5. После загрузки модели вы увидите сообщение «✅ GigaAM готов». Нажмите **F2** (или кнопку «Старт») и начинайте говорить.
