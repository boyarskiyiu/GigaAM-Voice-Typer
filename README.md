## 🚀 Возможности

- 🎤 **Распознавание русской речи** моделью GigaAM-v3 (700 тыс. часов обучения).
- 🧠 **Яндекс.Спеллер** — автоматическое исправление опечаток и повторов.
- 📦 **Полная автономность** — при первом запуске сама устанавливает все необходимые пакеты, ffmpeg и модель.
- 🎛 **Автоматический выбор микрофона** и калибровка уровня шума.
- 🔒 **Защита от повторного запуска** (lock-файл).
- ⏱ **Длинные фразы без потерь** — запись разбивается по паузам, всё распознаётся последовательно.
- 📊 **Плавный индикатор громкости** с цветовой индикацией.
- ⌨ **Горячие клавиши**:
  - `F2` — пауза / возобновить
  - `F3` — исправить последнюю фразу (открывается окно редактирования)
  - `F4` — свернуть окно в панель задач
- 🖥 **Компактный интерфейс** (820×730), всё помещается на экране.

## 📋 Требования

- **Windows 10/11** (программа ориентирована на Windows)
- **Python 3.8–3.11** (желательно 3.11)
- Доступ в интернет при первом запуске (для скачивания пакетов и модели)

## ⚙️ Установка и запуск

1. **Скачайте файл** `GigaAM_319.pyw` из этого репозитория.
2. Убедитесь, что у вас установлен Python (при установке отметьте галочку «Add Python to PATH»).
3. **Запустите двойным щелчком** по `GigaAM_319.pyw`.
   - Расширение `.pyw` скрывает окно консоли.
   - При первом запуске программа начнёт автоматическую установку недостающих пакетов и загрузку модели GigaAM (около 500 МБ). Это может занять несколько минут, в статусной строке будет отображаться прогресс.
4. После загрузки модели появится окно «Готов к работе». Начинайте говорить — текст будет автоматически вставляться в активное окно.

> **Примечание:** Если вы предпочитаете запускать через командную строку, используйте `python GigaAM_319.pyw`. Консоль при этом не появится.

## 🧰 Используемые технологии

- [GigaAM](https://github.com/sberdevices/gigaam) — модель распознавания речи от Сбера.
- [ONNX Runtime](https://onnxruntime.ai/) — движок для инференса нейросетей.
- [sounddevice](https://python-sounddevice.readthedocs.io/) — захват аудио с микрофона.
- [Tkinter](https://docs.python.org/3/library/tkinter.html) — графический интерфейс.
- [pyaspeller](https://github.com/konstantin612/pyaspeller) — клиент Яндекс.Спеллера.
- [keyboard](https://github.com/boppreh/keyboard) — глобальные горячие клавиши.

## 📁 Структура файлов

После первого запуска в папке со скриптом появятся:

- `corrections.json` — пользовательские исправления (через кнопку «Исправить»).
- `config.json` — сохранённые настройки (микрофон, чувствительность VAD — в данной версии не используются, но файл создаётся для совместимости).

## 🐞 Возможные проблемы и их решение

| Проблема | Решение |
|----------|---------|
| **Программа не запускается, окно сразу закрывается** | Запустите через командную строку, чтобы увидеть ошибку: `python GigaAM_319.pyw`. Скорее всего, не хватает прав на установку пакетов или нет интернета. |
| **Микрофон не найден** | Убедитесь, что микрофон подключён и работает в системе. Программа автоматически выбирает первое доступное устройство ввода. |
| **Текст не вставляется** | Проверьте, что активное окно поддерживает вставку (блокнот, Word, браузер). Попробуйте нажать `Ctrl+V` вручную после фразы. |
| **Индикатор громкости скачет** | Это нормально при тишине — фильтр сглаживает показания. При реальной речи индикатор должен реагировать адекватно. |
| **Ошибка `PermissionError` при удалении временного файла** | Программа автоматически повторяет попытку удаления. Если ошибка повторяется постоянно, перезапустите программу. |
