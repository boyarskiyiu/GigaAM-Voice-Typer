## 🚀 Возможности

- 🎤 **Распознавание русской речи** моделью GigaAM‑v3 (700 000 часов обучения).
- 🧠 **Яндекс.Спеллер** — автоматическое исправление опечаток и повторов.
- 📦 **Полная автономность** — при первом запуске сама устанавливает все необходимые пакеты, ffmpeg и модель.
- 🎛 **Автоматический выбор микрофона** и калибровка уровня шума.
- 🔒 **Защита от повторного запуска** (lock‑файл).
- ⏱ **Длинные фразы без потерь** — запись разбивается по паузам, всё распознаётся последовательно.
- 📊 **Плавный индикатор громкости** с цветовой индикацией.
- ⌨ **Горячие клавиши**:  
  - `F2` — пауза / возобновить  
  - `F3` — исправить последнюю фразу (открывается окно редактирования)  
  - `F4` — свернуть окно в панель задач
- 🖥 **Компактный интерфейс** (750×730), всё помещается на экране, шрифты крупные и читаемые.
- 🔄 **Автоматическое обновление** — при выходе новой версии программа сама скачает и установит её (достаточно нажать «Обновить»).

## 📥 Установка и запуск

1. Скачайте файл `GigaAM_Complete.pyw` из раздела [Releases](https://github.com/boyarskiyiu/GigaAM-Voice-Typer/releases/latest).
2. Убедитесь, что установлен **Python 3.8–3.11** (при установке отметьте «Add Python to PATH»).
3. Запустите файл двойным щелчком.
   - При первом запуске программа скачает модель GigaAM (~500 МБ) и установит все необходимые библиотеки. Это может занять несколько минут.
   - Расширение `.pyw` скрывает окно консоли.

> **Примечание:** Если вы предпочитаете командную строку, используйте `python GigaAM_Complete.pyw`.

## 🧰 Используемые технологии

- [GigaAM](https://github.com/sberdevices/gigaam) — модель распознавания речи от Сбера.
- [ONNX Runtime](https://onnxruntime.ai/) — движок для инференса нейросетей.
- [sounddevice](https://python-sounddevice.readthedocs.io/) — захват аудио с микрофона.
- [Tkinter](https://docs.python.org/3/library/tkinter.html) — графический интерфейс.
- [pyaspeller](https://github.com/konstantin612/pyaspeller) — клиент Яндекс.Спеллера.
- [keyboard](https://github.com/boppreh/keyboard) — глобальные горячие клавиши.
